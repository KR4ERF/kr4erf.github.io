Title: Flatpak Repository
Date: 2026-04-04

To install software from the KR4ERF repository, you must first add it to your system.

### 1. Add the Repository
Run the following command in your terminal:

```bash
flatpak remote-add --user --if-not-exists kr4erf https://dl.kr4erf.radio/kr4erf.flatpakrepo
```

### 2. Install Software
Once the repository is added, you can install apps provided by this repository.
